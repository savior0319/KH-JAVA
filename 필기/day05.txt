
	-----제어문-----

 @ Switch문
  - 변수의 값을 case 에서 찾아 명령을 수행하는 조건문
  - 해당하는 값이 없을 경우에에는 default의 명령문을 수행
  
  	* case와 명령문 사이에는 콜론 : 을 써야 함
 
 ex) 
 	Switch(변수){
 	
 	case 1 : 실행문1;
 			break;
 	case 2 : 실행문2;
 			break;
 	default : 실행문3; // else 와 같음
 	
 }
 
 
	-----반복문-----
 
 @ while 
 
  - 사용자가 원하는 만큼 명령들을 여러번 실행하는 제어 구조
  - 특정 조건이 만족되는 동안 명령을 계속적으로 실행 함
  - while문의 조건이 참일 경우에만 루프 처리를 반복 함
  		* 반복문을 처음 접할땐 초기, 조건, 증감 값을 생각하면 덜 어려움
  		
  		
 ### 표현식 ###
 
 while(조건문)
 {
 	실행문;
 }
