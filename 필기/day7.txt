
 @ 중첩 for 문
 - for문 안에 for문이 들어간 형태로 내부의 명령을 외부의 횟수만큼 반복하게 되는 구조

 
 for(초기값; 조건값; 증감값)
 {
 	for(초기값; 조건값; 증감값){
 		실행;
 	}
 }
 
 @ 무한 루프
 - 반복 횟수가 미리 정해져 있지 않고, 무한히 반복되는 루프
 
 ### 무한루프 생성코드 ###
 
 for문을 이용한 무한 루프
 
 for(;;){
 	실행문;
 }
 
 while문을 이용한 무한루프
 
 while(true){
 	실행문;
 }
 
 	@ 무한루프는 보편적으로 while문을 많이 사용 함
