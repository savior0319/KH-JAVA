 @ 배열
 - 배열은 동일한 타입을 가지는 변수들의 집합
 
 @ 배열의 쓰임
 - 여러 개의 변수를 한번에 선언 가능
 - 같은 형의 변수들을 연속적으로 쓸 수 있음
 
 @ 배열의 선언
 - 배열을 사용하기 위해서는 주소값을 저장하기 위한 래퍼런스 변수가 필요함
 	- 자료형 [] 변수이름;
 	- 자료형 변수이름[];
 	
 @ 배열의 초기화
 
    ex)
 	int [] arr = {1, 2, 3, 4, 5};
 	int [] arr = new int[]{1, 2, 3, 4, 5};
 	
 	에러 발생)
 	int [] arr = new int[5]{1, 2, 3, 4, 5};
